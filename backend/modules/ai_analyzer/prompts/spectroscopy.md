# 分析要点

1. **图谱概述**: 描述图谱的整体特征，包括信噪比、基线、峰的分布等
2. **关键峰归属**: 根据峰位置推断可能的官能团、晶相或化合物
3. **物质/结构推断**: 综合分析，推测可能的物质成分或结构特征
4. **数据质量评估**: 评估数据的可靠性，指出可能的问题
5. **进一步建议**: 建议可能需要的补充分析或实验

# 专业知识参考

## FTIR 常见峰归属
- 3200-3600 cm⁻¹: O-H 伸缩振动
- 2850-3000 cm⁻¹: C-H 伸缩振动
- 1650-1750 cm⁻¹: C=O 伸缩振动
- 1000-1300 cm⁻¹: C-O 伸缩振动

## XRD 分析要点
- 峰位置对应特定晶面
- 峰宽度与晶粒大小相关
- 峰强度比可用于相定量

## Raman 特征
- 与 FTIR 互补
- 对称振动信号强
- 水干扰小
